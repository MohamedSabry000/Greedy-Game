# Greedy-Game
-------------

* function that take a number and follow the below rule to convert it into dollars, quarters, dime, nickels and cents.

## Input
Number: 15.90



## Output
You have 15 dollar, 3 quarter, 1 dime, 1 nickel and 0 cent

## Rules
* 1 dollar = 100 cent
* 1 quarter = 25 cent
* 1 dime = 10 cent
* 1 nickel = 5 cent
